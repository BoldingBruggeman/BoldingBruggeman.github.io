Bolding & Bruggeman (BB)

BB develops and applies advanced numerical models for natual waters.

Bolding & Bruggeman ApS - BB - (until autumn 2014 Bolding & Burchard ApS) was established by Hans Burchard and Karsten Bolding in Varese, Italy 1999. Hans Burchard and Karsten Bolding were equal partners until 2014, when Jorn Bruggeman replaced Hans Burchard. BB is a research and development company specializing in numerical models for lakes and oceans.

Software

BB participates in a number of public software projects within the area of natural waters - notably GOTM and GETM - as well as our own Open Source projects. These are described on the portfolio page. The Fortran based software projects are available from Github and require a working Fortran compiler to be installed in order to compile the source code into a working program. The Python based projects are also hosted on Github - but in addition - also installable via pip via simple commands as e.g. pip install pyncview. Note that the software is compatible with Python 2.7 only. More information will be available on the project pages.

Projects

BB has over the years participated in a number of national in international research projects but do also participate in more applied projects. We are always interested to hear about potential collaboration options.
